
# Extract links from md v.1.0.0

This library will extract all links found in md file.

## Method

getLinksFromMd(text);

###### Example:

You can use Node.js:

```
$node
> const check = require("mark-markdown-down")
<<<<<<< HEAD
> check(readme.mdhere); // return links;
```
